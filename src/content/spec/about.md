## 关于博客

这个博客使用 Astro 静态网站生成器构建，采用 Fuwari 主题，托管在 GitHub 上，通过 Cloudflare 进行部署和加速。

## 技术栈

- 前端框架：Astro + Svelte
- 样式：Tailwind CSS
- 部署：Vercel / Cloudflare Pages
- 内容管理：Markdown
